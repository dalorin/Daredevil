Daredevil
Reddit Game Jam 05/10 submission
By Dalorin
================================

HOW TO PLAY
===========
Left Arrow, Right Arrow - Rotate ship
Up Arrow - Boost
Esc - Exit

AIM OF THE GAME
===============
Get points by flying dangerously close to planets. Beware their gravitational pull!
If you collide with a planet, it's game over.

DEPENDENCIES
============
python (developed with 2.6 - untested with other versions)
pygame 1.9.1
